O que podemos testar dentro de uma arquitetura de API rest?

Nós temos os usuários que utilizam, podem estar acessando um serviço de tv smart tv, serviço de streaming,  aplicativo mobille, página web, play jogos que ele acessa informações de jogos através de uma API, -> faz conexão com a internet -> e vai bater num servidor de back end para trazer as informações e fazer a validação das regras de negócios -> basicamente conecta no banco de dados para ver se a informação está lá, grava a informação, dependendo do tipo de operação. O back-end é quem cuida das APIS.
